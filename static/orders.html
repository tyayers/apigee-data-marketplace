Order Management API
1.0.0

An API for managing orders.

This is the documentation for version 1.0.0 of the API. Last update on Apr 12, 2024.

Protocol: kafka 
central.mykafkacluster.org:8092

Orders

Publish a new order

PUB orders

Publish a new order

PUBLISH orders

Message

Payload

  • ordertime number

    The time the order was placed.

  • id string

    The unique identifier of the order.

  • itemid string

    The item id of the order

  • total string

    The total amount of the order

  • status string

    The current status of the order.

    Values are created, processing, shipped, delivered, or cancelled.

Payload example 
{
  "ordertime": 42.0,
  "id": "string",
